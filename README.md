# 使用方法

下载解压 → 打开 weread-in-company.exe

可以看见熟悉的微信读书网页版了

最上面有一排六个按钮，功能分别是

拖动窗口、刷新、调整透明度、置顶窗口、最小化窗口（后台运行、托盘里有图标）、关闭窗口

接下来你要做的是

扫码登陆 → 调整窗口到合适的大小 → 调整透明度 → 置顶 → 开始摸鱼吧😊😊😊
